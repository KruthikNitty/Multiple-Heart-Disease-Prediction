# Multiple Heart Disease Prediction

This project predicts the likelihood of a person having heart disease based on medical attributes such as age, gender, chest pain type, blood pressure, cholesterol levels, and more. It helps in preventive screening using data-driven predictions.

---

## Features

- Input multiple health parameters
- ML-based prediction using medical dataset
- Web interface for easy data entry
- Fast and lightweight implementation
- Can be extended to support real model inference

---

##  Tech Stack

- HTML, CSS, JavaScript
- Machine Learning Dataset (Kaggle / UCI)
